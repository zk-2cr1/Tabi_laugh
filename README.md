# <旅ラフ>

## サイト概要
<p>・世界中の国の情報を共有できる
<p>・自分が訪れた国などの投稿を公開できる（ブログ風）
<p>・様々な人と交流ができる（コメント機能）

### サイトテーマ
<h5>世界一周した人、これからする人のためのコミュニティサイト

### テーマを選んだ理由
自分自身いつか世界一周の旅に出たいと思っているので、
<p>共通の思いがある人たち向けのコミュニティがあるといいなと思ったため

### ターゲットユーザ
世界一周旅に出た人、これから旅に出ようと思っている人

### 主な利用シーン
旅に出てる時
<p>旅に出る準備期間 etc...
